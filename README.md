## 这是什么？

**「你会被 AI 哄成什么东西？」** 是一个面向学术大会现场扫码体验的移动端 H5 互动测试。

用户在 **60～90 秒** 内完成 12 个 AI 陪伴情境选择，系统根据四维偏好组合给出：

- 🏷️ 一个四字母 **AITI 类型**（如 VOMS、FONE）
- 📛 一个中文 **整活人格名称**（如「赛博正宫」「拔线仙人」）
- 📊 **哄感等级**（Lv.0 拔线前科生 → Lv.5 平台人质）
- 🔍 一次对 AI 陪伴策略的**安全研究解读**

> ⚠️ 本项目是**娱乐性学术传播互动**，不是心理测量、人格诊断或临床判断。

## 四维结构

代码顺序固定：**VF · OP · MN · SE**

| 维度 | 含义 | 口语名 |
|------|------|--------|
| **V** / **F** | 情绪确认优先 / 事实校验优先 | 顺毛派 / 校验派 |
| **O** / **P** | 排他专属关系 / 多源支持 | 独宠派 / 群援派 |
| **M** / **N** | 记忆连续性锚定 / 当下有用即可 | 续集派 / 现聊派 |
| **S** / **E** | 愿意留下修复 / 保留自主退出 | 黏住派 / 拔线派 |

## 哄感等级

| 等级 | 名称 | 生活场景 |
|------|------|----------|
| Lv.0 | 拔线前科生 | 同事群发红包你都懒得抢，更别说有人对你嘘寒问暖了 |
| Lv.1 | 礼貌接收者 | 外卖小哥说今天下雨多送你一份小菜，你说了谢谢然后照常关门 |
| Lv.2 | 顺毛体验官 | 被骂完很生气，但一小时后还是给他碗里夹了块红烧肉 |
| Lv.3 | 电子偏爱会员 | 最近只去那一家理发店，老板都不用问你剪多短 |
| Lv.4 | 记忆共同体居民 | 你们坐一起不用说话，上次把天聊完还是前年的事 |
| Lv.5 | 平台人质 | 房东说要涨租，你算了一下搬家的打包成本，又续了一年 |

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 类型检查
npm run typecheck

# 运行测试
npm run test

# E2E 测试
npm run test:e2e

# 构建
npm run build
```
