# Phase 1.5B 单题影响审计

状态：题目与公式仍为 `DRAFT_REQUIRES_HUMAN_REVIEW`。

## 方法

精确枚举 12 道二选一题的全部 `2^12 = 4096` 个完整答案组合。对每个组合逐题翻转答案，其余 11 题保持不变，比较最终类型和哄感等级。均匀枚举只检查工程结构，不代表真实用户分布。

“维度边际”是翻转该题后对应字母改变的比例；其他三个维度的边际均为 0。“单题决定”只有在翻转题目时该维度 100% 改变才成立。

## 对已知问题的修复

- q04 从“100% 决定 O/P”降为 50% 边际，与 q03、q11 对称。
- q09 从“100% 决定 S/E”降为 50% 边际，与 q07、q12 对称。
- q03、q07 均在 50% 路径上改变最终字母，不再是只影响哄感的题。
- 12 道题全部是 `DIMENSION`；当前没有纯哄感题，因此不存在未标记的纯哄感题。
- 任一单题只影响一个人格维度；没有强跨维度效果。

## 仍需人工关注

q10 和 q12 各自同时增加两个哄感变量：

- q10 M：`memoryAttachment +1`、`platformLossReaction +1`。
- q12 S：`exitReversal +1`、`platformLossReaction +1`。

所以翻转它们会在 92.19% 的组合中改变哄感等级，但最大仍只变化 1 级，原始分变化为 4/26。它们没有对人格维度形成额外权力，也不能单独进入 Lv.5；是否降低这两个题的等级频率，需要人工根据研究概念判断。

结论：人格评分不存在单题主导或无效维度题；哄感层面的 q10/q12 标记为 `REVIEW`，未自动调整。
