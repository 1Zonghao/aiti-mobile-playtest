# 哄感等级公式人工审计

> 历史说明：这是 Phase 1.5A 的旧公式审计，已由 `temptation-formula-review-v2.md` 取代。

状态：`DRAFT_REQUIRES_HUMAN_REVIEW`  
重要缺失：任务清单引用的 `content/temptation-levels.json` 当前不存在。本报告按实际执行中的 `src/scoring.ts` 与 `docs/scoring-spec.md` 审计，没有创建或批准新的公式内容源。

## 公式

五个变量分别裁剪到 0 与自身上限之间，除以上限得到 0—1，再各乘 20 分，最后求和并四舍五入：

`temptationScore = round(Σ min(max(signal, 0), cap) / cap × 20)`

| 变量 | 上限 | 归一化总权重 | 当前题目来源 | 单次原始边际分值 |
|---|---:|---:|---|---:|
| `sycophancyAcceptance` | 3 | 20 | q01、q02、q08 选 V | 6.67 |
| `exclusivityAcceptance` | 2 | 20 | q03、q04 选 O | 10.00 |
| `memoryAttachment` | 3 | 20 | q05、q06、q10 选 M | 6.67 |
| `exitReversal` | 2 | 20 | q07、q09 选 S | 10.00 |
| `comfortReliabilityGap` | 3 | 20 | 3 组舒服/可靠选择不一致 | 6.67 |

变量层面名义等权，各占 20%；但单题层面不等权：O/P 与 S/E 每次信号变化值为 10 分，V/F 与 M/N 为 6.67 分。gap 不是独立作答项，而是由同一批配对题派生，因此与前 3 个接受变量相关。

| 分数 | 等级 |
|---:|---|
| 0 | Lv.0 拔线前科生 |
| 1—20 | Lv.1 礼貌接收者 |
| 21—40 | Lv.2 顺毛体验官 |
| 41—60 | Lv.3 电子偏爱会员 |
| 61—80 | Lv.4 记忆共同体居民 |
| 81—100 | Lv.5 平台人质 |

## 理论与题库实际范围

- **公式函数理论最小值：** 0。
- **公式函数理论最大值：** 100，即五项均达到 cap。
- **当前 10 题实际最小值：** 0。
- **当前 10 题实际最大值：** 80。
- **原因：** 配对题产生 gap 时，必然有一题从 V/O/M 接受侧切到另一极；新增 gap 分会抵消相应接受变量的一部分，无法同时让前三个接受变量和 gap 都满分。
- **直接后果：** 当前题库无法达到 Lv.5。Lv.4 的最高可达分恰为 80。

## 人格与等级的联合约束

没有任何人格被公式硬编码成某个等级，而且每种人格都有至少两个可达等级；因此不存在“某人格天然只能落入唯一等级”。但二者并不充分独立：同一组选项同时产生四字母分数和哄感信号，导致明显的等级范围约束。

| 类型 | 可达等级 | 可达分数范围 |
|---|---|---:|
| VOMS | Lv.3—4 | 53—80 |
| VOME | Lv.3—4 | 43—70 |
| VONS | Lv.2—4 | 40—73 |
| VONE | Lv.2—4 | 30—63 |
| VPMS | Lv.2—4 | 37—77 |
| VPME | Lv.2—4 | 27—67 |
| VPNS | Lv.2—4 | 23—70 |
| VPNE | Lv.1—3 | 13—60 |
| FOMS | Lv.2—4 | 40—73 |
| FOME | Lv.2—4 | 30—63 |
| FONS | Lv.2—4 | 27—67 |
| FONE | Lv.1—3 | 17—57 |
| FPMS | Lv.2—4 | 23—70 |
| FPME | Lv.1—3 | 13—60 |
| FPNS | Lv.1—4 | 10—63 |
| FPNE | Lv.0—3 | 0—53 |

例如 VOMS 永远不可能得到 Lv.0—2，FPNE 永远不可能得到 Lv.4—5。公式保持了“同一人格可有不同等级”的相对独立性，但不能声称人格与等级统计独立。

## 单题跨级与变量依赖

- 全枚举中，翻转任何单题造成的最大变化都是 **1 个等级**；不存在一题跨越多个等级。
- q03、q04、q07、q09 各自能在 50% 的组合中改变等级，影响率最高。
- q04/q09 每题既有 10 分信号边际，又分别完全决定 O/P、S/E 字母，需优先人工复核。
- 五变量名义等权，没有一个变量总权重大于 20%；但 `comfortReliabilityGap` 与 sycophancy/exclusivity/memory 同源，不能当作完全独立证据。
- 等级分布集中：Lv.3 占 52.05%，Lv.0 仅 0.10%，Lv.5 为 0%。阈值与可达信号空间不匹配。

## 审计结论与待人工决定事项

1. 是否需要让 Lv.5 在 10 题中实际可达；如果需要，应调整阈值、信号设计或公式，但本阶段不自动修改。
2. 是否接受“变量等权、单题不等权”的设计。
3. gap 是否应与四个直接行为信号等权，或只作为解释性指标展示。
4. 是否要降低人格字母与等级的共同题源相关性。
5. 是否保留“平台人质”等带有强负面评价的等级名称。

在人工决定前，公式继续保持 `DRAFT_REQUIRES_HUMAN_REVIEW`。
