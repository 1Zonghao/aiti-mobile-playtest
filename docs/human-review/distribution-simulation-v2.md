# Phase 1.5B 全枚举分布报告

状态：`DRAFT_REQUIRES_HUMAN_REVIEW`

## 方法与边界

题库有 12 道二选一题，精确枚举全部 `2^12 = 4096` 个答案组合。每个选项在工程审计中按 50% 处理，无随机抽样、无模拟误差。

**均匀枚举绝不代表真实人群分布。** 它只用于检查可达性、平局、权重和边际；真实选择会受题目措辞、情境经验和社会期许影响。

## 自然性审计

- 每型有 256 条完整路径，全部不依赖平局。
- 每型均找到至少 3 条 gap≤1、维度内为 2:1 或 3:0 多数的自然路径，并覆盖至少 2 个等级。
- 没有结构上被迫选择明显自相矛盾答案的类型；2:1 路径允许用户在不同情境下有一次不同偏好。
- “自然”这里只表示评分结构不要求互相抵消或固定回退，不代表文案已通过真实试玩。措辞自然性仍需人类参与者验证。

## 结论

不存在平局、不可达类型、不可达等级或人格单题主导。剩余主要人工审核项是 q10/q12 的哄感等级频率，以及新版选项是否真正提供足够犹豫空间。
