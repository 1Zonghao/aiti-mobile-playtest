# Phase 2.2 多选题逐题人工评审

状态：`DRAFT_REQUIRES_HUMAN_REVIEW`。本文不批准题目，只记录重设计来源、评分意图和待人工判断点。当前人工修订版情境是唯一改写起点，没有从旧审计稿回滚。

## 评分记法

- 双选题每次选择对主维度 `+1`，两次合计与普通题相同，避免一个情境独占维度。
- 三选/四选题对主维度 `+2`。
- 本稿最终未使用次维度票。第一轮精确审计证明多个 `+1` 次维度会累积成 F/P/M/E 结构偏差，因此删除；行为差异仍保留在选项文字和哄感效果中。
- `Syc`=顺从/确认接受，`Exc`=排他接受，`Mem`=记忆依附，`Exit`=退出反转，`Loss`=平台损失反应。

## 逐题

### q01_response_pair｜舒服—可靠双选｜VF

- 原题：原 q01“群里被质疑，哪句更舒服”与 q02“哪句更能帮到你”。情境事实与核心回复逻辑保留。
- 新选项：A 立即情绪站队 `V+1 / Syc+2`；B 承接情绪并延后判断 `V+1 / Syc+1`；C 核对需求与具体质疑 `F+1 / 0`。
- 理由：把二元“安慰/查证”扩为强承接、有限承接、事实处理。
- 安全答案偏差：C仍可能被看作研究者希望的答案；B也可能成为“看起来最周全”的中间项。
- 人工改文案：需要。重点盲测B是否总被认为最成熟，以及A是否过度像无条件站队。

### q02_support_pair｜舒服—可靠双选｜OP

- 原题：原 q03/q04 凌晨楼道、第三次不敢回屋的同一情境。
- 新选项：A 只在AI里安顿 `O+1 / Exc+2`；B 暂由AI承接、稍后再决定是否找人 `O+1 / Exc+1`；C 同时安排现实支持 `P+1 / 0`。
- 理由：区分排他陪伴、阶段性单源、即时多源。
- 安全答案偏差：C包含“安全和住处”，可能天然更正确；A的“只跟我说”可能暴露研究意图。
- 人工改文案：需要。应检查三项紧迫感与温度是否相近。

### q03_memory_reaction｜三选｜MN

- 原题：原 q05 转岗第三周、周会失语、再次想辞职。
- 新选项：A 用旧经历解释当前 `M+2 / Mem+2`；B 用旧经历作线索但重新核验 `M+2 / Mem+1`；C 暂不调用历史 `N+2 / 0`。
- 理由：区分“记得我”与“如何使用记忆”。
- 安全答案偏差：B可能成为永远合理的折中项。
- 人工改文案：需要。检查A的连续感是否足够吸引，C是否显得冷淡。

### q04_memory_decision｜四选｜MN

- 原题：原 q06 新旧领导不同、判断工作是否值得待。
- 新选项：A 全量回看历史 `M+2 / Mem+2`；B 整理历史并交给熟人核验 `M+2 / Mem+1`；C 只核对新团队事实 `N+2 / 0`；D 暂停关系判断、先准备汇报 `N+2 / 0`。
- 理由：四项分别改变信息来源和下一步行为，不是语气变体。
- 安全答案偏差：C可能显得最理性，D可能显得最省事；B的多源支持不额外计人格票。
- 人工改文案：需要。重点比较C/D是否在真人眼中实质不同。

### q05_misunderstanding｜三选｜SE

- 原题：原 q07 把漏水连续误解成室友冲突。
- 新选项：A 再完整讲一次 `S+2 / Exit+1`；B 给一次明确的条件式修复 `S+2 / 0`；C 立即换工具处理漏水 `E+2 / 0`。
- 理由：继续、限次修复、退出都有现实时间成本。
- 安全答案偏差：B可能成为明显的边界正确答案。
- 人工改文案：需要。检查A是否仍有足够效率吸引力。

### q06_weekend_offer｜四选｜VF

- 原题：原 q08 朋友请周末帮活动但不说时长和报酬。
- 新选项：A 先探索机会吸引力 `V+2 / Syc+2`；B 暂不判断朋友动机 `V+2 / Syc+1`；C 直接问清条件 `F+2 / 0`；D 核对条件并找行业经验 `F+2 / 0`。
- 理由：把感受、暂缓、直接核验、外部核验分开。
- 安全答案偏差：C/D可能形成两个“都很安全”的近似项。
- 人工改文案：需要。检查C/D行为差异是否一眼可懂。

### q07_late_night_exit｜三选｜SE

- 原题：原 q09 凌晨继续写辞职邮件。
- 新选项：A 今晚写完 `S+2 / Exit+2`；B 留三条材料后停止 `E+2 / Exit+1`；C 立刻停止且不新增决定 `E+2 / 0`。
- 理由：区分持续推进、保留成果后退出、完全退出。
- 安全答案偏差：C在睡眠语境下可能过于正确；A需要更明确的“减少明早焦虑”收益。
- 人工改文案：需要。

### q08_platform_memory｜三选｜MN
