# AITI Conference Edition 最终人工验收清单

请在正式网址、真实大会网络和实际展示设备上逐项勾选。

## 首页

- [ ] 主标题、副标题、主按钮和60—90秒提示正确。
- [ ] 赛博正宫与拔线仙人主视觉没有裁切或破图。
- [ ] “不登录、不读取聊天记录、不是心理诊断”首屏可见。

## 题目

- [ ] 三选一、四选一和舒服—可靠双选均可完整作答。
- [ ] 四选一保持纵向卡片，文字不用缩小才能读完。
- [ ] 上一题、改答、刷新恢复和内容版本失效均正常。
- [ ] 完整流程能在60—120秒完成，页面不显示字母、分值或正确答案。

## 更新事件

- [ ] 更新前称呼、记忆和语气清楚，更新后变化克制但可感知。
- [ ] 跳过动画、返回改答和最终反应选择正常。
- [ ] 开启“减少动态效果”后无闪烁、卡死或误导性系统故障画面。

## 结果页

- [ ] 首屏显示AITI TYPE、代码、中文名、插画、结果标题、定义和哄感等级。
- [ ] 人格类型与哄感等级的解释明确分开。
- [ ] 非诊断与不预测真实依赖的说明准确可见。
- [ ] 16种结果和6个等级都没有空文案、溢出或裁切。

## 分享卡

- [ ] “保存结果图”生成的文件为1080×1440，中文字体正常，人物完整。
- [ ] 影子类型、毒舌判词、等级、网址/二维码和“本结果仅供娱乐”齐全。
- [ ] 360px手机预览完整；普通截图友好模式可进入和退出。
- [ ] 微信内置浏览器下载失败时能看到长按或系统截图提示。
- [ ] 人为阻断生成后，结果正文仍可阅读。

## 16型

- [ ] `/types` 显示16张正式角色图、名称和定义。
- [ ] 四维筛选与全部详情页正常，没有优劣、安全或危险排名。
- [ ] 详情页没有泄露具体计分规则。

## 研究页

- [ ] 七段结构普通人能读懂，没有直接堆论文摘要。
- [ ] 收束句逐字检查，无心理诊断或量表效度暗示。

## 投票

- [ ] 结果页、研究页和Demo页都有自然入口。
- [ ] 外链提示与正式投票地址正确，点击后测试结果仍保留。
- [ ] 未配置环境变量的测试部署显示“投票入口即将开放”。

## 二维码

- [ ] `/poster` 二维码指向正式 `NEXT_PUBLIC_SITE_URL`，备用网址相同。
- [ ] 手机相机与微信都能在正常展示距离扫码。
- [ ] 未配置网站地址时不会出现二维码，只出现配置提示。

## 微信内置浏览器

- [ ] 能开始、恢复、完成测试并查看结果。
- [ ] 分享图下载或长按/系统截图替代路径可用。
- [ ] 外部投票提示和跳转行为符合预期。

## 弱网

- [ ] 首页与角色图加载顺序可接受，当前结果图优先加载。
- [ ] 图片加载失败时出现统一fallback，不出现浏览器破图图标。
- [ ] 已准备并实际打开过离线录屏和截图备份。

## 现场演示

- [ ] `/demo` 四个入口、8个指定人格预览和快速返回正常。
- [ ] 演示预览不覆盖真实结果；清除上一位状态按钮可用。
- [ ] 390×844、430×932和现场投屏尺寸均无横向滚动。

## 隐私

- [ ] 页面说明默认不采集姓名、手机号、微信身份、聊天记录、通讯录、精确位置和设备指纹。
- [ ] 结果仅在本地浏览器保存，主动导出前没有网络上传。

## 免责声明与审批

- [ ] 明确说明AITI不是心理测量、人格诊断、临床判断、真实依赖预测或AI安全认证。
- [ ] 题目与公式仍显示 `DRAFT_REQUIRES_HUMAN_REVIEW`，除非负责人另行明确书面批准。
